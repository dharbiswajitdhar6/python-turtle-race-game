# Turtle Race Game 

A simple turtle racing game built using Python Turtle Graphics.

## Features
- User can bet on a turtle color.
- Six turtles race randomly.
- Random movement decides the winner.

## Technologies Used
- Python
- Turtle Module

## How to Run

```bash
python main.py
```
